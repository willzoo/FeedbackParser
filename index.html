Instructions

  1. Click "Choose Excel File" to select your Excel file.
  2. The file will be processed based on the following rules:
    • Row 0: Contains questions (one per column pair).
    • Row 1 onwards: Data rows with names and scores.
    • Column 0: Ignored.
    • Before column AR:
      • Odd columns (1, 3, 5, ...): Scores.
      • Even columns (2, 4, 6, ...): Names.
    • From column AR onwards:
      • Even columns (AR, AT, AV, ...): Feedback sentences.
      • Odd columns (AS, AU, AW, ...): Names.
    • Columns BF-BH: Scores only (no names), each column is one question.
    • Column BI: Sentences only (no names).
    • Names: Full names are used for matching/display.
    • Questions: The string ' or NA' is ignored from the end if present.
    • Questions that are identical after removing ' or NA' are combined.
  3. Results will display the counts of all unique scores/feedback given by each person for each question.
